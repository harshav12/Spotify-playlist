# Music Playlist Web App

## Description
A basic music playlist web application built using HTML, CSS, and JavaScript to practice front-end development fundamentals and JavaScript event handling.

The project implements a simple audio player with a predefined playlist and basic playback controls.

## Features
- Play and pause audio tracks
- Next and previous song controls
- Seek bar to control playback position
- Predefined playlist with limited songs
- Simple and clean user interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Deployment
The project is deployed using GitHub Pages and can be accessed through the live demo link.

## Limitations
- The playlist contains a fixed number of songs
- The layout is not fully responsive across all screen sizes

## What I Learned
- JavaScript DOM manipulation
- Handling audio playback using JavaScript
- Implementing user interactions like play, pause, next, and seek
- Structuring a small front-end project for deployment
